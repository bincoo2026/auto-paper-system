# 粘贴内容自动转换公式格式计划

## 问题描述

在编辑题目对话框中，当用户粘贴内容时，如果内容包含 `\(...\)` 或 `\[...\]` 格式的公式，这些格式不能被 @aarkue/tiptap-math-extension 正确识别和渲染。需要自动将其转换为：
- `\(...\)` → `$...$` （行内公式）
- `\[...\]` → `$$...$$` （块级公式）

## 修改计划

### 步骤1：定位编辑器初始化代码
- 编辑器在 `index.html` 文件中的 `window.loadEditors` 函数内初始化
- 使用 `createEditor` 函数创建三个编辑器（stemEditor、answerEditor、analysisEditor）
- 编辑器使用了 Tiptap 和 @aarkue/tiptap-math-extension

### 步骤2：添加 paste 事件处理
在编辑器初始化后，为每个编辑器添加 paste 事件监听器：

```javascript
// 为每个编辑器添加 paste 事件处理
stemEditor.on('paste', (event) => {
  handlePaste(event, stemEditor);
});
answerEditor.on('paste', (event) => {
  handlePaste(event, answerEditor);
});
analysisEditor.on('paste', (event) => {
  handlePaste(event, analysisEditor);
});
```

### 步骤3：实现 handlePaste 函数
创建 `handlePaste` 函数来处理粘贴事件：

```javascript
function handlePaste(event, editor) {
  // 获取剪贴板数据
  const clipboardData = event.clipboardData;
  if (!clipboardData) return;

  const text = clipboardData.getData('text/plain');
  if (!text) return;

  // 转换公式格式
  let convertedText = text;

  // 将 \(...\) 转换为 $...$
  convertedText = convertedText.replace(/\\\(([^)]+)\\\)/g, '$$$1$$');

  // 将 \[...\] 转换为 $$...$$
  convertedText = convertedText.replace(/\\\[([\s\S]+?)\\\]/g, '$$$$$1$$$$');

  // 阻止默认粘贴行为
  event.preventDefault();

  // 插入转换后的内容
  editor.commands.insertContent(convertedText);
}
```

- 文件：`/home/zhao/project/auto-paper-system/frontend/index.html`
- 位置：在 `window.loadEditors` 函数中，编辑器初始化完成后（约第873行附近）
- 在 `export { stemEditor, answerEditor, analysisEditor };` 之前添加 paste 事件监听器

### 注意事项
1. 转换顺序很重要：先转换块级公式 `\[...\]` 再转换行内公式 `\(...\)`
2. 使用非贪婪匹配 `[\s\S]+?` 来匹配跨行内容
3. 阻止默认粘贴行为，使用 `insertContent` 插入转换后的内容

## 预期效果
- 当用户在编辑器中粘贴包含 `\(...\)` 格式的内容时，自动转换为 `$...$`
- 当用户粘贴包含 `\[...\]` 格式的内容时，自动转换为 `$$...$$`
- 转换后的公式能够被 @aarkue/tiptap-math-extension 正确识别和渲染
