$\lim_{x \to \infty } \frac{\sin x}{x} =1$.
答案：错误
解析：0

---

$\lim_{x \to 0 } \frac{\sin x}{x} =1$.
答案：正确
解析：第一个重要极限，或者等价无穷小。

---

$\lim_{x \to 0 } x\sin \frac{1}{x} =1$.
答案：错误
解析：0，无穷小乘以有界量。

计算 $\lim_{x \to 0} \frac{x - \sin x}{x^3} = \underline{\qquad\qquad}.$
答案：$\frac{1}{6}$
解析：该极限为 $\frac{0}{0}$ 型，连续三次应用洛必达法则：   $$\lim_{x \to 0} \frac{x - \sin x}{x^3}<br>= \lim_{x \to 0} \frac{1 - \cos x}{3x^2}<br>= \lim_{x \to 0} \frac{\sin x}{6x}<br>= \lim_{x \to 0} \frac{\cos x}{6} = \frac{1}{6}.$$   （最后一步也可用等价无穷小 $\sin x \sim x$ 直接得 $\frac{1}{6}$。）
